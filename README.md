🎓 Student Grade System (Java Swing GUI)

A simple and interactive desktop application built with Java Swing that allows users to enter a student’s name and marks, then calculates and displays the corresponding grade with proper validation and error handling.

🚀 Features
🧑 Enter student name
📊 Input marks (0–100)
🎯 Automatic grade calculation (A, B, C, D, F)
⚠️ Input validation with error handling
🖥️ Clean and user-friendly GUI (Java Swing)
💬 Pop-up error messages for invalid inputs

🧠 Grade Logic
Marks Range	Grade
80–100	A
70–79	B
60–69	C
50–59	D
Below 50	F


🛠️ Technologies Used
Java ☕
Swing (JFrame, JLabel, JTextField, JButton)
AWT (Event Handling, Layout, Colors)
OOP Concepts
Exception Handling
